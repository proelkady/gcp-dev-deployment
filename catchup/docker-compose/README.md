# Catchup Dev Deployment

## Components 

- Nginx: Used as a reverse proxy for routing traffic to different services.
- MongoDB: A document-oriented database used for storing and retrieving data.
- MySQL: A relational database management system used for structured data storage. Works for Wordpress.
- Wordpress: A content management system for creating and managing websites.
- Keycloak: An authentication and authorization server
- PostgreSQL: A powerful, open-source relational database system.
